# EdgeNat怎么样？EdgeNat介绍和测评

## EdgeNat 介绍
EdgeNat成立于2019年，专注于提供VPS云服务器和独立服务器，覆盖美国、韩国和香港等多个数据中心。EdgeNat的产品以高性价比和多样化的线路选择著称，特别是其美国双ISP住宅IP、三网AS4837线路以及香港、韩国的三网直连和CN2线路。
